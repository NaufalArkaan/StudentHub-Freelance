===========================================================
STUDENTHUB - MARKETPLACE FREELANCER MAHASISWA
===========================================================

Nama Proyek :
StudentHub - Marketplace Freelancer Mahasiswa

Deskripsi :
StudentHub merupakan platform marketplace berbasis web yang
mempertemukan mahasiswa sebagai penyedia jasa (Freelancer)
dengan mahasiswa maupun pengguna lain sebagai pencari jasa
(Client). Sistem juga menyediakan halaman Admin untuk
mengelola pengguna, kategori jasa, order, dan laporan.

===========================================================
TECH STACK
===========================================================

Framework  : Next.js (App Router)
Language   : TypeScript
Database   : Supabase PostgreSQL
Authentication : Supabase Auth
Styling     : Tailwind CSS
ORM         : Supabase Client
Package Manager : npm

===========================================================
MINIMUM REQUIREMENTS
===========================================================

- Node.js v20 atau lebih baru
- npm
- Git
- Browser (Google Chrome / Microsoft Edge)

===========================================================
CARA MENJALANKAN SISTEM
===========================================================

1. Ekstrak folder project.

2. Buka terminal pada folder project.

3. Install seluruh dependency.

   npm install

4. Buat file .env.local kemudian isi dengan konfigurasi
   Supabase.

   NEXT_PUBLIC_SUPABASE_URL=xxxxxxxxxxxxxxxx
   NEXT_PUBLIC_SUPABASE_ANON_KEY=xxxxxxxxxxxxxxxx

5. Jalankan aplikasi.

   npm run dev

6. Buka browser.

   http://localhost:3000

===========================================================
BUILD PRODUCTION
===========================================================

Build aplikasi

npm run build

Menjalankan hasil build

npm start
